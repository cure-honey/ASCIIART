# ASCIIART
make ASCII ART from BMP file

元プログラムを、fortran66 さんのここからこころよく提供いただきました。
http://fortran66.hatenablog.com/entry/2014/11/28/034535
付け加えた点は、読み込める BMP サイズを任意幅にしたところと、白地、黒地の部分に変化をつけるようにしたことです。
